# Data Organization in Spreadsheets for Social Scientists

This is the GitHub repository for the "Data Organization in Spreadsheets for Social Scientists" lesson, part of the [Data Carpentry - Social Science curriculum](https://datacarpentry.org/lessons/#social-science). To see the lesson content go to [http://datacarpentry.github.io/spreadsheets-socialsci/](http://datacarpentry.github.io/spreadsheets-socialsci/)

## Contributing

We'd love to know if you have suggestions for improving this lesson!

You can do this by submitting an [issue](https://github.com/datacarpentry/spreadsheets-socialsci/issues) in this repo, sending an email to the current maintainers listed below, or reach out on our [Slack channel](https://carpentries.slack.com/messages/C9X34DJ9Z).

If you want to know more about contributing to this lesson and other Carpentries efforts, please read the [CONTRIBUTING guide](CONTRIBUTING.MD).

## About the data for the lesson

You can find more information about the data used for the workshop on the [Social Science Workshop Overview lesson](https://datacarpentry.github.io/socialsci-workshop/data.html).

This lesson in particular uses three files: SAFI_clean.csv, SAFI_messy.xlsx, and SAFI_dates.xlsx, which you can download in the [Data section](https://datacarpentry.github.io/spreadsheets-socialsci/#data) of the lesson.

You can also read the [DATA_README.md](DATA_README.md) file, with descriptions of the data and fields in each of these three files.
